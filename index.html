    ศึกษา

    โรงเรียนทุ่งศุขลาพิทยา “กรุงไทยอนุเคราะห์"
    วิทยาลัยเทคโนโลยีวิศวกรรมแหลมฉบัง

    สวัสดีครับ ผมชื่อ ธีรภัทร สังวาลเดช กำลังศึกษาอยู่ที่วิทยาลัยเทคโนโลยีวิศวกรรมแหลมฉบัง สาขาเทคโนโลยีสารสนเทศ (Information Technology) ผมมีความสนใจเป็นพิเศษในด้านการพัฒนาเว็บไซต์ (Web Development) การออกแบบระบบฐานข้อมูล (Database Design) และการพัฒนาระบบโปรแกรมเพื่อใช้งานจริง โดยมีความรู้พื้นฐานในการเขียนโปรแกรมภาษา HTML, CSS และ JavaScript และกำลังศึกษาเพิ่มเติมเกี่ยวกับการพัฒนาเว็บไซต์แบบ Full Stack นอกจากนี้ ผมยังมีความสนใจด้านการออกแบบกราฟิก และการสร้างโมเดล 3 มิติ (3D Modeling) โดยเฉพาะสไตล์ Anime ซึ่งช่วยพัฒนาความคิดสร้างสรรค์ และทักษะด้านการออกแบบของผมให้ดียิ่งขึ้น ผมเป็นคนที่ชอบเรียนรู้สิ่งใหม่ ๆ และพัฒนาตนเองอยู่เสมอ พร้อมที่จะเรียนรู้เทคโนโลยีใหม่ ๆ เพื่อเพิ่มทักษะและประสบการณ์ในสายงาน IT เป้าหมายของผมในอนาคตคือการเป็น Web Developer หรือ Full Stack Developer ที่มีความสามารถในการพัฒนาระบบเว็บไซต์ที่มีประสิทธิภาพ ใช้งานง่าย และตอบโจทย์ผู้ใช้งานได้อย่างแท้จริง ผมมีความมุ่งมั่น ตั้งใจ และพร้อมที่จะพัฒนาตนเองเพื่อเติบโตในสายงานด้านเทคโนโลยีสารสนเทศต่อไป
